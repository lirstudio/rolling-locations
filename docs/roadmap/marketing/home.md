# Marketing Page – Home

Back to [Marketing Phase](../phases/05-marketing-website.md)

## Route

`/`

## Objective

Create a high-conversion homepage for creators and hosts with clear discovery and signup paths.

## Sections

1. Navbar
2. Hero
3. Category Highlights
4. How It Works
5. Featured Locations
6. Social Proof / Testimonials
7. CTA Banner
8. Footer

## Section requirements

### 1) Navbar
- Logo + product name
- Primary links (Discover, How it works, For hosts)
- Language switch (HE/EN UI-ready)
- Primary CTA (Sign up)

### 2) Hero
- Clear value proposition
- Search starter controls (city/category placeholders)
- Primary CTA + secondary CTA
- Hero visual/illustration placeholder

### 3) Category Highlights
- Grid/carousel of featured categories
- Each card links to locations index with filter preselected

### 4) How It Works
- 3 steps for creators
- 3 steps for hosts (optional tabbed variant)

### 5) Featured Locations
- Reusable location cards (mock data)
- “View all locations” CTA

### 6) Social Proof
- Testimonials cards
- Optional counters (hosts, bookings, cities)

### 7) CTA Banner
- Host recruitment message
- Primary CTA to sign-up with host preselection

### 8) Footer
- Core links
- Legal links placeholders
- Social links placeholders

## Data mode

Frontend mock data only.

## Done when

- [ ] All sections implemented
- [ ] Mobile + desktop responsive
- [ ] RTL-friendly spacing and alignment
- [ ] One clear primary CTA hierarchy
