# Marketing Page – 404

Back to [Marketing Phase](../phases/05-marketing-website.md)

## Route

Global not-found page (`src/app/not-found.tsx`)

## Objective

Provide a branded, helpful fallback page that keeps users in the product journey.

## Sections

1. Error Headline
2. Context Message
3. Recovery Actions
4. Visual Element

## Section requirements

### 1) Error Headline
- Clear “Page not found” heading
- Keep tone helpful, not technical

### 2) Context Message
- Explain that the URL may have changed
- Keep copy concise

### 3) Recovery Actions
- Primary action: back to home
- Secondary action: go to locations index

### 4) Visual
- Lightweight branded illustration/icon

## Done when

- [ ] Page is branded (not generic template text)
- [ ] Primary + secondary navigation actions exist
- [ ] Works in RTL and mobile layouts
