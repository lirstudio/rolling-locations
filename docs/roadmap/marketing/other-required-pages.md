# Marketing – Other Required Pages

Back to [Marketing Phase](../phases/05-marketing-website.md)

This file defines pages that are not in the initial must-build list but are usually required for a production-ready marketing website.

## Priority A (recommended for v1)

### 1) About
- Route: `/about`
- Purpose: brand story, trust, who this is for

### 2) Contact
- Route: `/contact`
- Purpose: lead/contact form and support channels

### 3) FAQ
- Route: `/faq`
- Purpose: reduce support load and clarify booking process

## Priority B (can be post-v1)

### 4) For Hosts
- Route: `/for-hosts`
- Purpose: dedicated host acquisition landing page

### 5) Terms & Privacy
- Routes: `/terms`, `/privacy`
- Purpose: legal baseline for public launch

## Minimal section template for each new page

1. Header
2. Main content blocks
3. Supporting CTA
4. Footer

## Decision rule

If a page is required by compliance or acquisition goals, create it in v1.
If not, keep in backlog and prioritize core conversion flow first (Home → Locations Index → Location Details → Booking intent).
