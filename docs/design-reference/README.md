# Design reference

Inspiration images and mockups for Rollin Locations UI. **Agents:** read [INDEX.md](./INDEX.md) first to choose the right reference, then use the Read tool on the image file. Always adapt to Rollin design tokens (see `docs/project-specification.md` and the rollin-design-tokens skill).

- **Add new images:** Place PNG/JPG/WebP here and add a row to `INDEX.md` with suggested use.
- **Filenames:** Prefer descriptive names; long Dribbble-style names are fine if indexed in INDEX.md.
