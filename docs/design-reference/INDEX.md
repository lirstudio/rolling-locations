## Agent workflow

1. **Before implementing a screen/component:** Check this index for the best-matching reference.
2. **Load the image:** Use the Read tool on the chosen file (e.g. `docs/design-reference/Property Rental Booking Website by Conceptzilla on Dribbble.png`).
3. **Adapt, don’t copy:** Apply layout, hierarchy, and patterns while keeping Rollin design tokens (primary red, neutrals, shadcn components).
